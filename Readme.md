# NVIDIA Stock Performance Prediction Web Tool
# Overview

This project is an interactive financial analytics web application built using Flask, Plotly, and Machine Learning.
It analyzes NVIDIA’s historical stock data and predicts future performance trends using a Linear Regression model.
The web app offers dynamic visualizations and analytical insights for better understanding of stock trends and volatility.

# Features

- Stock Trend Forecasting using Linear Regression
- Interactive Plotly Graphs for actual vs. predicted visualization
- Moving Average Overlay for volatility analysis
- Automatic Metrics Computation (RMSE, R², Correlation, etc.)
- Responsive Flask Dashboard Interface

# Tech Stack

Frontend: HTML, CSS, Plotly.js

Backend: Flask (Python)

Libraries: Pandas, NumPy, Scikit-learn, Plotly, SQL lite

Deployment: Render

**Skills Demonstrated**
- Python (Pandas, NumPy, Scikit-learn)
- API Integration (`yfinance`)
- SQL (via SQLite)
- Data Visualization (Matplotlib)
- Financial Feature Engineering
- Machine Learning (Linear Regression)
